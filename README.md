# 🎓 Udemy Clone (HTML & CSS)

A simple **Udemy Clone** built using only **HTML and CSS**.  
This project replicates the design and layout of the Udemy online learning platform, focusing purely on front-end structure and styling without any JavaScript functionality.

---

## 🚀 Features

- 🎨 Clean and responsive design  
- 🏠 Home page with featured courses  
- 📚 Course listing layout  
- 👨‍🏫 Instructor section  
- 📞 Footer with links and contact info  
- 💡 Pure HTML & CSS — no frameworks or JS  

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3 (Flexbox, Grid, and basic animations)**
